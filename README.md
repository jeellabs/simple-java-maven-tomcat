# simple-java-maven2025
# Simple Java Maven Tomcat Project

This project demonstrates building a Java web application using Maven and deploying it on Apache Tomcat.

## Tech Stack
- Java 17
- Maven 3.9
- Apache Tomcat 9

## Build
mvn clean 
mvn package

## Deploy
Copy WAR file to Tomcat webapps directory.
